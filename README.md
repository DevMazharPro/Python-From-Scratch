# Python Learning Journey

A comprehensive collection of Python programming exercises and tutorials structured chapter-by-chapter for progressive learning.

## üìö Course Structure

### Chapter 1: Introduction to Python

- **01_first.py** - Your first Python program
  - Hello World implementation
  - Introduction to comments (single-line and multi-line)
  - Basic usage of external libraries (`pyjokes`)
- **01_first_practice.py** - Practice exercises for Chapter 1

### Chapter 2: Python Fundamentals

- **02_Variables.py** - Variable declaration and assignment
- **02_DataTypes.py** - Core Python data types
  - Integers, Strings, Floats, Booleans, None
  - Variable naming conventions and best practices
- **02_Types.py** - Type system exploration
- **02_Input.py** - User input handling
- **02_Operators.py** - Mathematical and logical operators
- **02_practice.py** - Comprehensive practice exercises for Chapter 2

### Chapter 3: String Manipulation & F-Strings

- **03_Intro_Strings.py** - Introduction to string basics
- **03_escape_seq_characters.py** - Escape sequences and special characters
- **03_fstring.py** - Comprehensive f-string tutorial
  - Basic f-string syntax and usage
  - Expression embedding and calculations
  - Number formatting (decimals, percentages, scientific notation)
  - Text alignment and padding
  - Dictionary and object formatting
  - Date and time formatting
  - Advanced features and debugging techniques
- **03_str_functions.py** - Built-in string methods and functions
- **03_practice.py** - Practice exercises for Chapter 3

### Chapter 4: Data Structures - Lists & Tuples

- **04_list.py** - Introduction to Python lists
  - List creation and indexing
  - Mutable nature of lists
  - Mixed data types and nested lists
- **04_list_methods.py** - Comprehensive list operations
  - Adding elements (append, insert)
  - Removing elements (remove, pop, clear)
  - Sorting and reversing
  - Slicing and searching
  - Counting and finding elements
- **04_tuple.py** - Introduction to Python tuples
  - Immutable sequences
  - Tuple creation and access
  - Tuple unpacking and mixed data types
- **04_tuple_methods.py** - Tuple operations and methods
  - Built-in tuple methods (count, index)
  - Tuple concatenation and repetition
  - Membership testing
  - Converting between lists and tuples
- **04_practice.py** - Practice exercises for Chapter 4
  - User input collection with lists
  - Sorting student marks
  - List operations and tuple counting
